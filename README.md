# InHouse

InHouse is a Next.js app for connecting customers with personal chefs.

## Local development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   npm run dev
   ```
3. Open:
   `http://localhost:4028`

## Scripts

- `npm run dev` — start local dev server on port 4028
- `npm run build` — production build
- `npm run start` — dev server shortcut
- `npm run serve` — serve production build
- `npm run lint` — run linting
- `npm run lint:fix` — auto-fix lint issues
- `npm run format` — format source files
- `npm run type-check` — TypeScript checks
